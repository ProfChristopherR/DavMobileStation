# DAV Mobile Station - Liceo San Nicolás

Estación meteorológica móvil desarrollada para el **Liceo de Excelencia Polivalente San Nicolás**. 

Este proyecto visualiza datos en tiempo real y tendencias históricas obtenidos desde un servidor de ArcGIS.

## Características
- **Monitoreo en Vivo**: Temperatura, Humedad y Velocidad del Viento.
- **Gráficos Dinámicos**: Visualización de tendencias históricas con ECharts.
- **Filtro de Fecha**: Consulta de datos históricos por días específicos.
- **Diseño Premium**: Interfaz moderna con Tailwind CSS y efectos de cristal (Glassmorphism).

## Instalación y Uso
1. Descarga los archivos del repositorio.
2. Coloca tu logo en la carpeta raíz con el nombre `logo.png`.
3. Abre el archivo `index.html` en cualquier navegador moderno.
